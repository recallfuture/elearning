# 查作业

为华北理工大学的教学系统做的快捷查作业的App，目的是方便学生了解还有需要交的作业有哪些。

## 特色

- 界面简单明了
- 记住密码，下次自动登录
- 自动根据作业的截止日期设定定时通知
